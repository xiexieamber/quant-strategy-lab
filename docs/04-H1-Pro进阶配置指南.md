## Step 2：中证1000 均线择时（防风格切换）

### 在哪改

页面中部 → **大盘择时**

### 怎么点

1. 选择 **自定义择时公式**（部分账号显示 VIP 试用，免费用户通常也能回测）
2. 在公式框粘贴：

```
if(指数收盘(000852)>MA(指数收盘(000852),20),1,0)
```

### 含义（通俗版）

| 情况 | 仓位 |
|------|------|
| 中证1000 收盘价 > 20 日均线 | **100% 满仓** 做小市值 |
| 否则 | **0% 空仓**（资金进黄金 ETF） |

### 可选：半仓版（更温和）

```
if(指数收盘(000852)>MA(指数收盘(000852),20),1,0.5)
```

弱势时仍保留 50% 仓位，不完全空仓。

### 指数代码

| 指数 | 代码 | 用途 |
|------|------|------|
| 中证1000 | **000852** | 中小盘代表（推荐） |
| 中证2000 | **000932** | 更小盘（可选对比） |

### 注意

- 择时控制的是 **整个策略仓位**，不是单只股票
- 公式在 **调仓前一日** 计算，调仓日生效（[果仁帮助](https://guorn.com/stock/help)）

## Step 3：最低成交额筛选（防流动性陷阱）

### 在哪改

**择股设置 → 筛选条件** → 添加一条

### 推荐配置（三选一先试）

| 方案 | 指标 | 条件 | 值 | 适合 |
|------|------|------|-----|------|
| **A（推荐）** | **成交金额** 或 **当日成交额** | 大于 | **5000000**（500万） | 平衡 |
| B（更严） | 同上 | 大于 | **8000000**（800万） | 更大票、更稳 |
| C（更松） | 同上 | 大于 | **3000000**（300万） | 保留更多迷你盘 |

### 在果仁里怎么搜

搜索框输入：**成交金额** 或 **成交额** → 选「大于」→ 填数字（单位：元）

### 含义

去掉「每天成交额太少、一卖就崩」的票。2024 年微盘踩踏时，流动性差的票跌最惨。

### 可选进阶

若界面有 **20日平均成交额**，可设：

```
20日平均成交额 > 5000000
```

比「只看当天」更稳定。

## Step 4：ROE / 盈利为正（防业绩雷）

### 在哪改

**择股设置 → 筛选条件** → 再添加一条

### 推荐配置（三选一）

| 方案 | 指标 | 条件 | 值 |
|------|------|------|-----|
| **A（推荐）** | **净利润** | 大于 | **0** |
| B | **扣非净利润** | 大于 | **0** |
| C | **基本每股收益** | 大于 | **0** |

### 若有「净资产收益率 ROE」

```
净资产收益率 > 0
```

或更严：

```
5年平均ROE > 10
```

（ROE 数据可能空值较多，空值股票会被排在后面或筛掉）

### 含义

小市值 + 亏损股 = 业绩雷高发区。加这条是「小盘里再选赚钱的」，不是纯赌最小。

### 注意

加盈利筛选后，**年化可能下降**，但 **2024 类暴雷年** 往往更稳。属于用收益换安全。

## Step 5：单票止损（可选）

### 在哪改

**交易模型 II → 卖出条件** → 添加一条

### 方法 A：界面有「止损」选项（最简单）

| 设置 | 值 |
|------|-----|
| 止损 | **-12%** 或 **-10%** |

含义：单只股票亏到 12% 就卖，防一只票拖垮组合。

### 方法 B：自定义公式（界面有「自定义」时）

在 **卖出条件** 添加公式：

```
后复权收盘价 - $beginPrice < 0 - 0.12 * $beginPrice
```

约等于：从买入价算，跌超 12% 卖出。

### 方法 C：最长持有天数（脚本已验证可用）

| 设置 | 值 |
|------|-----|
| 持有天数 | **≥ 60** 或 **≥ 100** |

含义：不管排名如何，持有超过 N 天就卖，防止死拿变雷。

### 卖出条件之间是「或」关系

已有「排名 ≥ 15 就卖」，再加止损 = **满足任一条件就卖**。

## 预期现象（心里有数）

| 步骤 | 可能效果 |
|------|----------|
| Step2 择时 | 回撤↓，但可能错过反弹；2024 初可能少亏 |
| Step3 成交额 | 回撤略↓，年化略↓ |
| Step4 盈利>0 | 年化明显↓，暴雷股减少 |
| Step5 止损/持有天数 | 换手↑，单票大亏减少 |

**不要追求每一步都提高年化**；目标是 **最大回撤和 2024 年表现** 改善。
