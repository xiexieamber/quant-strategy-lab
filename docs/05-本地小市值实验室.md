# 本地小市值实验室

用 **聚宽 JQData** 在本地跑「果仁 H1」小市值策略，带网页界面调参，**不依赖果仁 VIP**。

## 快速启动

```bash
# 1. 确保 .env 里已配置 JQ_USERNAME / JQ_PASSWORD
# 2. 安装依赖（含 streamlit）
pip install -r requirements.txt

# 3. 启动界面
python3 scripts/run_small_cap_lab.py
# 或
streamlit run apps/small_cap_lab/app.py
```

浏览器会自动打开（默认 http://localhost:8501）。

## 策略说明（对应果仁 H1）

| 项目 | 默认值 |
|------|--------|
| 排名 | 流通市值↑ 权重 2，总市值↑ 权重 1 |
| 模型 II | 排名 ≤8 买入，≥15 卖出 |
| 持股 | 最多 10 只，等权 |
| 调仓 | 每 1 个交易日，**开盘价** |
| 空仓替代 | 黄金 ETF `518880` |
| 成本 | 单边 0.2% |

## 本地额外支持（果仁需 VIP 或难批量）

- **1/4 月空仓**：预设「H1 + 1/4月空仓」
- **中证1000 MA20 择时**：预设「H1 + 中证1000择时」
- **成交额 ≥ 500 万、扣非净利润 > 0、止损 12%**：预设「H1 Pro（全风控）」

## 数据限制

试用 JQData 账号通常只有 **约 1 年** 历史数据。界面顶部会显示你账号的实际可用区间。

回测按**每个交易日**拉一次全市场估值，1 年大约需要 **3～8 分钟**，请耐心等待进度条。

### 聚宽每日 100 万条额度（JQData 专用）

试用账号每天最多查询约 **100 万条** 数据。若使用 **AkShare**，则不受此限制。

本项目已做优化：
- **批量拉取** + **本地缓存**（`data/processed/jq_cache/` 或 `ak_cache/`）
- 同一天内反复调参，**几乎不再消耗额度**
- JQ 额度若今天拉不完，**明天再点「开始回测」** 会自动续拉

界面顶部可选 **AkShare（推荐）** 或 **JQData**。

## 代码结构

## 命令行回测（不用界面）

```python
from src.strategies.small_cap.config import SmallCapConfig
from src.backtest.small_cap_engine import run_small_cap_backtest

cfg = SmallCapConfig(start="2025-03-01", end="2026-03-01")
result = run_small_cap_backtest(cfg)
print(result.annual_return, result.max_drawdown)
```

## 与果仁的差异

- ✅ 小市值排名 + 模型 II 买卖逻辑
- ✅ 1/4 月空仓、均线择时、成交额/净利润筛选（本地免费）
- ❌ 预期 ST、审计意见等果仁专有指标（暂未接入）
- ❌ 多年历史（受 JQData 账号权限限制）
